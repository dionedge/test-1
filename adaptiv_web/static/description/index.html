A carefully crafted experience. Customizable by you.

Completely rewritten backend theme for Odoo. The Adaptiv backend theme is dedicated to providing the best Odoo experience possible.

Get the most out of your screen area

Meet the shell overlay

The shell overlay replaces every large dialog found in Odoo and displays content on the side for an optimized experience. At small screen sizes the shell panes will completely cover the screen.

Truly mobile optimized

Navigation left and right

On mobile devices you can quickly switch between apps using the left hand menu. The right hand menu allows you to navigate between screens.

Make it yours

Multi company branding

You know which colors you love the most. That's why the Adaptiv backend theme gives you full control over it's color palette.

The company branding feature allows multi-company installations to retain their unique look. Each company can have up to four different logo formats:

  • Both a regular and an inverse logo can be provided.
  • The inverse logo is shown in the sidebar while the regular one is used by Odoo elsewhere.
  • Use the svg file option for sharp looking logo's on any screen size.
  • Use the fallback png option in case this is not possible.

Compatibility

Browser: Google chrome >=50, Firefox >=52, Safari >=10, IE >= 11, Edge >= 15
Odoo: 11.0 (tested against commit a0a5443e8f0606b561d2955449a690ea8a178703)
Odoo modules: All official Odoo modules. 3rd party modules with custom css might cause trouble.

Installation

This theme has several additional modules which are automatically installed, depending on your installed Odoo features. Be sure to download these additional modules (at no extra costs) to ensure a working installation:

adaptiv_mail

Bridge module between adaptiv_web and mail.
adaptiv_web_editor

Bridge module between adaptiv_web and web_editor.
adaptiv_web_tour

Bridge module between adaptiv_web and web_tour.
adaptiv_website

Bridge module between adaptiv_web and website.
adaptiv_portal

Bridge module between adaptiv_web and portal.
adaptiv_web_settings_dashboard

Removes branding from the settings dashboard. (Optional)
